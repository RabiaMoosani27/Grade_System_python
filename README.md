## 📊 Student Grade System (Python)
## 🧾 Overview

The Student Grade System is a Python-based beginner project that calculates a student's total marks,
percentage, and final grade based on input marks. It helps understand basic programming concepts like conditions,
arithmetic operations, and functions.

## 🚀 Features
Input marks for subjects
Individual subject grades
Total marks
Percentage
Final overall grade

## 🧠 Logic Used
📌 Step 1: Total Marks

Total marks are calculated by adding all subject marks.

📌 Step 2: Percentage

P=(Total Marks/Obtained Marks)×100
📌 Step 3: Grade System
90 – 100 → A+
80 – 89 → A
70 – 79 → B
60 – 69 → C
50 – 59 → D
Below 50 → Fail
