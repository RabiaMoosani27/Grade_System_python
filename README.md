## 📊 Student Grade System (Python)
## 🧾 Overview

The Student Grade System is a Python-based beginner project that calculates a student's total marks, percentage, and final grade based on input marks. It helps understand basic programming concepts like conditions, arithmetic operations, and functions

## 🧾 Project Overview

This Python program calculates:

Individual subject grades
Total marks
Percentage
Final overall grade

## 🧠 How It Works
1. Individual Subject Grade

Each subject mark is checked separately and a grade is assigned:

90–100 → A+
80–89 → A
70–79 → B
60–69 → C
50–59 → D
Below 50 → Fail

## 🎯 Learning Outcomes
Arithmetic operations in Python
Conditional statements (if-elif-else)
Functions (optional improvement)
Real-world logic building
